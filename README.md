# AI-Based Classroom Disturbance Detection & Student Voice Monitoring System

## Overview
An intelligent classroom management system that uses AI to monitor student voice activity, detect disturbances, and generate comprehensive reports for educators.

## Features
- 🔐 JWT-based Authentication (Students & Lecturers)
- 🎤 Voice Sample Upload & Analysis
- 📊 Real-time Disturbance Detection
- 📈 Interactive Charts (Pie, Bar, Line graphs)
- 🎯 Disturbance Scoring Algorithm
- 📝 Automated Report Generation
- 🎨 Modern Black Theme with Neon UI
- 📱 Fully Responsive Design
